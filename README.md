# pokemon-statistical-analysis
Aplicación de estadística descriptiva y visualización de pokémones.
## **objetivos**
-Comprobar que si Los pokemones legendarios realmente son mas fuertes.
-Comprobar que tipo tiene mejores estadisticas promedio. 
-Comprobar si los pseudolegendario son realmente competitivos.
-Comprobar si influye el Egg group en el poder.
-Evolucion del poder por generación.
